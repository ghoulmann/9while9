Here...

...we put into practice what we learn from the communities at TurnKey Linux and Raspberry Pi.

Following TurnKey Linux's brilliant model, we work with TKLDEV to produce server appliances that lower the barriers to getting what you want and need up and running.

Simultaneously, we work to make setting up those same technologies as "turn key" as possible on Raspbian.

Ghost

Our first foray into TKLDEV: Ghost is a revolutionary new blogging platform that relies on markdown syntax. We've successfully produced a Ghost appliance for submission to TKL.

Ghost is built to run with nodejs; using forever and Nginx, we think we've got this ready for use. The ISO produced by TKLDEV uses sqlite and runs Ghost in production mode.

Ampache

Ampache is web-based, on demand audio and video streaming software. Years ago, our first attempt to produce an appliance for submission to TKL used TKLPatch to get Ampache up and running with as little configuration. WIth the release of TKLDEV, we've ported the patch and relied on the wisdom we've accumulated over some years to make the appliance truly turn key.

Scratching

Meanwhile, middle division students in the lab are learning loops, variables, and the importance of commenting code using MIT's Scratch. Given this base code, they're making customizations and improvements.

Using the cursor keys, try to maneuver the red spot at the start to the yellowish spot at the finish without running into the maze walls. Using the mouse, click the flag to begin.

A Call for Help

In the age of Common Core State Standards, clearly none of this is sanctioned by the state's department of education. About a year ago, we began a curriculum proposal; while states scurried to implement Common Core standards, there was not, of course, anyone who'd pay attention to this proposal-in-progress.

Consequently, we've got a few more months to revise and elaborate the proposal before there's another opportunity to ask someone to consider it. In the meantime, we're still looking for collaborators enthusiastic about what Computing at School has done for ICT curriculum reform in the UK. Please dm @9while9 on Twitter if you'd like to collaborate.
